# NayePankh Foundation Internship Management System

## Project Overview

This project is a Streamlit-based Internship Management System developed for NayePankh Foundation. It helps manage internship applications efficiently through a modern web interface.

## Features

* Applicant Registration
* SQLite Database Integration
* Dashboard Analytics
* Search Applicants
* Filter by Internship Domain
* Delete Applicants
* CSV Report Generation
* Form Validation
* Gmail Validation
* 10-Digit Phone Number Validation
* Duplicate Email Prevention

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas

## How to Run

Install dependencies:

pip install streamlit pandas

Run the application:

streamlit run app.py
